# Referee list generation

A TypeScript Node.js process which retrieves referees data from Leverade and saves it in two files.
One with public data only, and the other including some private data.

## Requirements

Node.js, npm and nvm

## Setup

Install and use the appropriate version of Node.js

```
nvm install
```

Install dependencies

```
npm install
```

## Run the process

```
npm start
```
